# spring-boot-validation
Spring Boot validation and template with a form   - Udacity Java Nanodegree.

## Tools used :

    https://start.spring.io/
    Spring Boot2.1.8.RELEASE
    Spring Web
    Thymeleaf
    DevTool
    Maven
    Embedded Tomcat
    
 ## Run on command line  
 
    mvn spring-boot:run 
    
 ## In browser - Result
 
     localhost:8080/add
     Result:  Add User form (username, password, grade, email)
